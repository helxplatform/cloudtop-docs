Cloudtop is a docker implementation of a virtual desktop that can be run on a headless cloud instance. CloudTop uses a combination of Apache Guacamole, TigerVNC and XFCE to provide web based access to a Linux XFCE desktop that can run on any platform that can run docker, including MacOS and any cloud based virtual machine instance.
